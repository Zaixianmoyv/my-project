## 介绍

> 项目 基于Vue3 + SpringBoot + Mysql的图书管理系统

## 功能

### 前台

​ 1、用户登录（读者）

​ 2、用户注册（读者）

​ 3、登录成功后，跳转首页

​ 4、在首页可以浏览相关图书信息

​ 5、在图书搜索栏页面可以借阅书籍以及按书名查询书籍

​ 6、在个人中心页面可修改个人信息

​ 7、在消息界面可接收由后台管理员发送的信息

​ 8、退出登录

### 后台

​ 1、用户登录（管理员）

​ 2、登录成功后，跳转首页

​ 3、在管理员页面登录管理员进行添加、修改、查询、禁用的功能

​ 4、在读者管理页面可禁用读者以及按姓名查询读者信息

​ 5、在借阅管理页面可以按姓名查询以及以及删除相关借阅人信息

​ 6、在图书管理页面可以进行添加、修改、查询、删除的功能

​ 7、在图书类型管理页面可以进行添加、修改、查询、删除的功能

​ 8、在公告管理页面管理员可以进行添加公告的操作

​ 9、退出登录

## 技术选型

### 前端项目技术栈

- Vue3

- Vue-router
- ElementPlus

### 后端项目技术栈

- SpringBoot

- Node.js
- Mysql
- MybatisPlus

## 项目初始化

#### 前端项目初始化步骤

1. 安装 Vue 脚手架
2. 通过 Vue-Cli 创建项目
3. 配置 Vue-router
4. 配置 ElementPlus组件库
5. 配置 Axios 库

#### 后端项目的环境安装配置

​ 1.安装MySQL数据库（数据库连接名：root 密码： 123456）

​ 2.配置项目相关信息
将数据库文件夹内的book.sql导入数据库中
将books文件夹放到D盘目录下

​ 3.启动项目
VScode: npm run serve
IDEA:run

​ 4.使用Apifox测试后台项目接口是否正常
